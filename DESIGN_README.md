## 📋 Design System Overview

### Colors
```
Primary:    #3B82F6 (Blue - actions)
Success:    #10B981 (Green - approved)
Warning:    #F59E0B (Yellow - caution)
Danger:     #EF4444 (Red - errors)
Background: #0B0E14 (Deep dark)
Surface:    #151A22 (Cards)
Text:       #FFFFFF (Main)
Muted:      #94A3B8 (Secondary)
```

### Components
- **Buttons:** Primary, Secondary, Outline, Success, Danger (5 variants)
- **Cards:** Base card with hover effect
- **Forms:** Input fields with focus states
- **Alerts:** Success, Warning, Danger, Info (4 types)
- **Navigation:** Sticky navbar with logo
- **Tables:** Striped with hover
- **Badges:** Status indicators
- **Modals:** Centered dialog boxes
- **Tabs:** Tabbed interfaces
- **Pagination:** Page navigation

### Typography
- **Font:** Inter (all weights)
- **H1:** 2.5rem | **H2:** 2rem | **H3:** 1.3rem
- **Body:** 1rem | **Small:** 0.875rem
- **Line height:** 1.6 (readable)
